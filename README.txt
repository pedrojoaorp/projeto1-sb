O compilador desenvolvido para a máquina hipotética neste projeto pode ser executado da seguinte forma após compilado ou aproveitando o arquivo executável presente neste diretório: 

Windows: 
./compilador arquivo.asm

Linux: 
./compilador arquivo.asm

OBS: Após execução do compilador, os resultados estarão nos arquivos .pre, o1 e o2 deste diretório. 

Foram utilizadas e recomendadas as seguintes versões do compilador GCC para uso deste projeto: 

Windows: 
gcc version 6.3.0 (MinGW.org GCC-6.3.0-1)

Linux: 
gcc (Ubuntu 7.5.0-3ubuntu1~18.04) 7.5.0
